Estruturas condicionais

Em JavaScript, estruturas condicionais são usadas para executar diferentes blocos de código com base no fato de uma determinada condição ser verdadeira ou falsa. Em resumo é uma estrutura de domada de decisões, temos duas estruturas condicionais que são muito importante: IF e SWITCH.

SWITCH

Uma condicional estrutural switchem JavaScript é usada para testar a igualdade de uma expressão com um valor especificado em cada case, e executar as instruções correspondentes se houver uma correspondência.

No primeiro caso, testamos se fruta é igual a 'Banana'. Se for, imprimimos a mensagem 'A fruta é uma Banana'no console. O break é usado para sair do switch assim que a correspondência for encontrada. No segundo case, testamos se fruta é igual a 'Maçã'. Se for, imprimimos a mensagem 'A fruta é uma Maçã' no console. No terceiro case, testamos se fruta é igual a 'Melão'. Se for, imprimimos a mensagem 'A fruta é um Melão' no console. Não default, testamos se nenhum dos casos anteriores teve correspondência. Se nenhum deles tiver correspondência, imprimimos a mensagem 'Não conheço essa fruta' no console.
